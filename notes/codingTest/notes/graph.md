# 그래프 이론

## 그래프(Graph)란?

- 그래프는 노드와 노드 사이에 연결된 간선(edge)의 정보를 가지고 있는 자료구조이다.

|                | 그래프                  | 트리                |
| -------------- | ----------------------- | ------------------- |
| 방향성         | 방향 또는 무방향 그래프 | 방향 그래프         |
| 순환성         | 순환 및 비순환          | 비순환              |
| 루트 노드      | 루트 노드 없음          | 루트 노드 존재      |
| 부모-자식 관계 | 부모-자식 관계 없음     | 부모-자식 관계 존재 |
| 모델 종류      | 네트워크 모델           | 계층 모델           |
